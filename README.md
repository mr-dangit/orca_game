# Orca Hunt

A browser-based Three.js game where you hunt as an orca, crack ice floes, follow the compass to prey, and unlock a pink portal into a unicorn meadow world at level 2.

## Run Locally

Open `index.html` in a browser, or host the folder with any static web server.

## Deploy on Vercel

Import this GitHub repository into Vercel as a static project. No build command is required.
